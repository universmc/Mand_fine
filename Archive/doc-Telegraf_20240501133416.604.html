Présentation

Ce projet consiste à créer une série de vidéos TikTok de 68 secondes chacune, présentant un personnage utilisant des lunettes AR et un drone virtuel. Le script comporte cinq scènes et inclut des fichiers multimédias associés.

Utilisation de GitHub

Pour obtenir le code source et les fichiers multimédias associés, veuillez cloner le dépôt GitHub à l'adresse suivante : https://github.com/votre-utilisateur/votre-dépôt

Script

Le script du projet est accessible depuis le fichier 'script.json'. Il présente une description détaillée de chaque scène et renvoie vers les fichiers multimédias correspondants.
